# Instafilter
학습용 SwiftUI 토이프로젝트입니다.

# 사용된 기술
- onChange modifier
- ConfirmationDialog(누르면 sheet처럼 View가 아래서 생성된다, 그후 Picker처럼 옵션을 선택)
- CoreImage
- 앨범에 이미지 저장하기, 앨범의 이미지 선택하기
  - UIImageWriteToSavedPhotosAlbum
  - ImgaePicker
- SwiftUI에서 UIKit사용하기
  - UIViewControllerRepresentable
  - Coordinator 디자인 패턴
